Go.iO - a Go push server
=====

A Go server providing a push server using sockjs and rabbitmq/redis.

![Go.iO Sequence Diagram](https://raw.github.com/snormore/go.io/master/resource/sequence.png)

Bootstrap
=========
source sbin/init-env.sh
sbin/bootstrap.sh

or do it yourself:
source sbin/init-env.sh
wget vagrant.dmg
